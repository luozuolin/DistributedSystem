##分布式系统
+   定义
    +   将不同的组建分布在不同的服务器上，给用户提供一个可靠统一的服务
    +   基本原则（CAP）
        +   C 一致性
        +   A 可用性
        +   P 分区容错性
    +   基本原则（BASE）
        +   基本可用
        +   软状态
        +   最终一致性
+   几种分布式系统
    +   多数据源
    +   多服务
    +   SOA
    +   微服务架构
+   分布式系统需要考虑的问题
    +   服务拆分
    +   数据拆分
    +   计算分拆
    +   服务状态以及异常处理
+ 微服务架构组成
    +   服务发现和注册
    +   服务网关与负载均衡
    +   监控和熔断机制
    +   配置消息等
+   分布式事务原则
    +   强一致性
    +   弱一致性
    +   最终一致性
